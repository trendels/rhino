class TestClient

Wraps a WSGI application under test.

Provides helpers to pass requests to the application and inspect the response.

__init__(app)

options(path_info, environ=None, **kw)

post(path_info, body, content_type=None, environ=None, **kw)

Send a POST request to the application.

If body is a dictionary, it will we submitted as a form with content_type=‘application/x-www-form-urlencoded’. Otherwise, the content_type parameter is required.

put(path_info, body, content_type, environ=None, **kw)

Send a PUT request to the application.

request(method, path, environ=None, **kw)

Send a request to the application under test.

The environment will be populated with some default keys. Additional keyword arguments will be interpreted as request headers.

For example, passing x_foo=1 will add a request header “X-Foo: 1”.
